# 🛍️ Retail Sales Analysis using SQL

## 📌 Project Overview
This project analyzes a retail sales dataset using SQL Server. It demonstrates SQL skills by solving real-world business problems through data exploration and analysis.

## 🎯 Objectives
- Analyze retail sales performance
- Explore customer purchasing behavior
- Identify top-performing product categories
- Generate business insights using SQL queries

## 🛠️ Tools Used
- SQL Server Management Studio (SSMS)
- SQL Server

## 📂 Dataset
The dataset contains retail transaction records with the following fields:
- Transaction ID
- Sale Date
- Sale Time
- Customer ID
- Gender
- Age
- Category
- Quantity
- Price per Unit
- Cost of Goods Sold (COGS)
- Total Sale

## 📊 SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions (COUNT, SUM, AVG)
- CASE Statements
- Date & Time Functions
- Window Functions (RANK)

## 📈 Business Questions Answered
- Total number of sales transactions
- Number of unique customers
- Sales by product category
- Average age of customers by category
- High-value transactions
- Monthly sales trends
- Top 5 customers by total sales
- Best-selling product categories
- Sales by shift (Morning, Afternoon, Evening)
